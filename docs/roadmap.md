# Product Roadmap

## MVP

- Multi-tenant auth
- Gratitech tenant profile
- Manual opportunity entry
- Grants.gov ingestion
- Transparent scoring
- Pipeline dashboard
- Opportunity detail page
- Proposal workspace
- Reusable proposal library
- Stripe trial and portal hooks
- Resend notification hooks
- Super-admin dashboard
- Docker Compose deployment

## Version 1 Hardening

- Stripe webhook reconciliation
- Team invitations and magic link login
- Deadline and trial notification schedules
- CSV import
- Full-text search indexes
- File upload validation and MinIO storage
- DOCX/PDF proposal exports
- Audit coverage for all sensitive actions
- More tenant isolation tests

## Version 2 Scaffold

- SAM.gov contract opportunities module
- Contract fit scoring
- NAICS/PSC matching
- Past performance library
- Capture management workflow
- Teaming partner database
- RFP compliance matrix data model
- Proposal color team review data model

## Version 2 Hardening

- CRM integrations
- Foundation and corporate connectors
- AI proposal drafting assistant
- Organization-specific knowledge base
- Attachment/package retrieval from SAM.gov
- RFP parsing into compliance matrices
- Color team review permissions and notifications
