# GrantAtlas.ai Brand Assets

The production logo system is code-native SVG, not raster text.

## Files

- `apps/web/public/brand/grantatlas-mark.svg`: app icon and compact mark
- `apps/web/public/brand/grantatlas-logo.svg`: horizontal logo for light backgrounds
- `apps/web/public/brand/grantatlas-logo-light.svg`: horizontal logo for dark backgrounds
- `apps/web/public/favicon.svg`: browser favicon
- `apps/web/components/BrandLogo.tsx`: React logo component used by the marketing site and app shell

## Visual Direction

The mark combines:

- atlas/map contour lines
- a navigation compass route
- an upward funding/capture signal

Primary colors:

- Navy: `#102033`
- Electric blue: `#37A2FF`
- Teal: `#42D7BE`
- Deep teal: `#0F766E`
- Light ink: `#F8FBFF`

Use the full wordmark in headers and decks. Use the mark alone for favicons, app icons, and compact navigation.
